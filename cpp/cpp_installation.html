1. Simple

C++ is a simple language because it provides a structured approach (to break the problem into parts), a rich set of library functions, data types, etc.


2. Portability

C++ is a portable language and programs made in it can be run on different machines. It is also known as Machine Independent.


3. Fast

C++ programs tend to be compact and run quickly. Hence the compilation and execution time of the C++ language is fast.


4. Compiler Based

C++ is a compiler-based programming language, which means no C++ program can be executed without compilation. C++ compiler is easily available, and it requires very little space for storage. First, we need to compile our program using a compiler, and then we can execute our program.


5. Dynamic Memory Allocation

When the program executes in C++ then the variables are allocated the dynamical heap space. Inside the functions, the variables are allocated in the stack space. Many times, We don't know how much memory is needed to store particular information in a variable soo the size of required memory can be expands dynamicaly at run time.


6. Recursion

In C++, we can call the function within the function. It provides code reusability for every function. This is very Interesting topic you can explore it and experiment it by your self for Better Understanding.
We will see it in Depth Later in this Tutorial.


7. Object Oriented Programming

In C++, object-oriented concepts like data hiding, encapsulation, and data abstraction can easily be implemented using keyword class, private, public, and protected access specifiers. Object-oriented makes development and maintenance easier.


8. Rich Library

C++ provides a lot of inbuilt functions that make the development fast.
Some of these are:

  • < iostream >
  • < cmath >
  • < cstdlib >
  • < fstream >

9. Mid-Level Programming

C++ is a programming language that combines both low-level and high-level features, earning it the title of a mid-level or intermediate programming language. It is commonly employed for creating system applications like kernels and drivers.


10. Case Sensitive

C++ is a programming language that pays attention to uppercase and lowercase letters, which means it's case-sensitive. For instance, using "cin" is correct for taking input from the input stream, but using "Cin" won't work. In contrast, some other languages like HTML and MySQL are not case-sensitive; they don't care if you use uppercase or lowercase letters in the same way.
