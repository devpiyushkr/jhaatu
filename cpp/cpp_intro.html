Introduction

C++ is a powerful and flexible programming language that evolved from the original C programming language. Created by Bjarne Stroustrup in the early 1980s, C++ is widely used in various fields due to its versatility and efficiency. The "++" in C++ symbolizes the language's evolutionary nature, signifying improvements over its predecessor, C.


C++ finds applications in diverse domains, ranging from system programming to game development, embedded systems, high-performance computing, and more. It is the language behind many software systems, game engines, operating systems (e.g., Windows), and critical applications requiring efficient resource management.


What is C++

C++ is an object-oriented programming (OOP) language, meaning it supports the creation and manipulation of objects, allowing for modular and reusable code. It extends the procedural programming features of C, adding object-oriented concepts like classes and inheritance. This blend of procedural and object-oriented paradigms makes C++ suitable for a broad range of applications.

C++ is a middle-level language, as it encapsulates both high and low level language features.


Efficiency: C++ is known for its emphasis on performance and efficiency, making it a preferred choice for resource-intensive tasks.


Portability: It can run on various platforms with minimal modifications, enhancing cross-platform development.


Standard Template Library (STL): C++ comes with the STL, a collection of tools that simplify common programming tasks, such as managing data structures and algorithms.


Community and Standards: C++ has a vibrant community of developers, and its standards are continuously evolving, ensuring consistency and compatibility across different systems.


Object-Oriented Programming (OOPs)

C++ supports the object-oriented programming, the four major pillar of object-oriented programming (OOPs) used in C++ are:

1. Inheritance

2. Polymorphism

3. Encapsulation

4. Abstraction
